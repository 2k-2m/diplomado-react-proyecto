# Gestión de Tareas - Diplomado React

Aplicación de lista de tareas desarrollada con React 19, TypeScript y Material UI.

## Funcionalidades
- Login y registro de usuarios
- Crear, editar y eliminar tareas
- Cambiar estado entre Pendiente y Finalizada

## Tecnologías
- React 19
- TypeScript
- Material UI
- Axios
- React Router DOM
- Zod
